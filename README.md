# General Wave Behavior Along Uniform Guiding Structures

This repository contains notes, explanations, and diagrams related to wave propagation in uniform guiding structures such as transmission lines, waveguides, and optical fibers.

## Topics Included
- Wave equations
- Boundary conditions
- TE, TM, and TEM modes
- Transmission lines
- Waveguides
- Field distributions
- Power flow and attenuation
